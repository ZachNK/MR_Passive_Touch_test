# MR_Passive_Touch_test
-AR및 MR환경에서 시각 및 청각적 자극으로 가상촉감 구현 연구를 위한 실험물.

-J. J. Gibson에 따른 능동 촉감 (Active Touch)과 수동 촉감 (Passive Touch)으로 촉감 유형 구분.

-S/W: Unity, H/W: HoloLens사용.

-상기 콘텐츠는 본 실험용이 아닌, 파일럿 테스트를 위한 실험 자극물임.
